# Inventory Management System - Frontend

A modern, responsive dashboard for inventory management built with HTML, CSS, and JavaScript. This frontend connects to a FastAPI backend to provide real-time inventory tracking and analytics.

## Live Demo

**Live Application**: https://inventory-management-frontend-ivory.vercel.app

**Backend API**: https://inventory-management-backend-2-2h44.onrender.com

## Features

-  **Interactive Dashboard** - View total products, inventory value, and low stock alerts
-  **Product Management** - Add, view, and delete products
-  **Transaction Recording** - Record stock IN/OUT movements with notes
-  **Real-time Analytics** - Automatic inventory valuation and transaction history
-  **Clean UI** - Modern, responsive design that works on all devices
-  **Live Updates** - Dashboard refreshes automatically after each action

## Tech Stack

- **HTML5** - Structure and content
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactive features and API integration
- **FastAPI Backend** - RESTful API for data management
- **Vercel** - Frontend hosting and deployment

## Dashboard Features

The dashboard displays:
- **Total Products** - Count of all products in inventory
- **Inventory Value** - Total value of all stock (price × quantity)
- **Low Stock Alerts** - Products below reorder threshold
- **Recent Transactions** - Last 5 stock movements
- **Product List** - Complete inventory with stock levels
